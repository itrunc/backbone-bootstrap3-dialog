# backbone-bootstrap3-dialog
Bootstrap3 Dialog with Backbone View

## Installation

* Install node modules

  ```bash
  npm install
  ```

  You might use sudo before npm.

* Install components

  ```bash
  bower install
  ```

* Build

  ```bash
  gulp
  ```

* Run demo

  ```bash
  node demo
  ```

  Access http://localhost:3000/demo/
