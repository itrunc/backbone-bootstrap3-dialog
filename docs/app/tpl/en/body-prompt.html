Prompt

You can call the prompt function to show a prompt dialog

BootstrapDialog.prompt(function(content) {
  BootstrapDialog.alert(content);
});

Prompt dialog with default content

BootstrapDialog.prompt('My name is panben', function(content) {
  BootstrapDialog.alert(content);
});

Prompt dialog with option

var promptOption = {
  title: 'Input your name',
  type: BootstrapDialog.TYPE.INFO,
  content: 'panben',
  okLabel: 'Completed',
  cancelLabel: 'Close',
  callback: function(content) {
    BootstrapDialog.alert('Hello '+content);
  }
};
BootstrapDialog.prompt(promptOption);

promptOption

NameTypeDefaultDescription
title string 'Input Content' The title of the dialog
content string '' The content you will input
type string BootstrapDialog.TYPE.PRIMARY
okLabel string 'OK' Label of the OK button
cancelLabel string 'CANCEL' Label of the cancel button
callback function null
