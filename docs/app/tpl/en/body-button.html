Button

Options:

NameTypeDefaultDescription
label string 'OK' Label on the button
type string The same as modal option type
icon string '' Refer to the Glyphicons component of bootstrap
cssClass string '' CSS class name for the button
category string BootstrapDialog.BUTTON_TYPE.BUTTON button type: button/submit/reset
size string BootstrapDialog.BUTTON_SIZE.NORMAL button size
block boolean false if true, the button is a block box, else, it is inline
disabled boolean false if true, the button is disabled
preventDefault boolean false if true, the click event will return false
action function null

function(self, dialog) {...}

Example

#code
[{
  label: 'I am a button',
  type: Bootstrap.TYPE.PRIMARY
}, {
  label: 'I am another button',
  type: Bootstrap.TYPE.SUCCESS',
  disabled: true
}]
