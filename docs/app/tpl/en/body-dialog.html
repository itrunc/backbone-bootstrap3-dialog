Dialog Instance

You can call the create function to create an dialog instance

BootstrapDialog.show();

Dialog with more options

BootstrapDialog.show({
  title: 'Hello Dialog',
  message: '<div class="alert alert-info">I am a dialog</div>'
});

Dialog defines type

BootstrapDialog.show({
  title: 'Hello Dialog',
  message: 'I am a dialog',
  type: BootstrapDialog.TYPE.INFO
});

Dialog can't be closed by click outside

BootstrapDialog.show({
  title: 'Hello Dialog',
  message: 'I am a dialog',
  backdrop: false
});

Dialog defines width

BootstrapDialog.show({
  title: 'Hello Dialog',
  message: 'I am a dialog',
  width: '60%'
});

Dialog defines buttons, See Button for more options of button

var options = {
  title: 'Please Choose',
  message: 'What is your sex?',
  type: BootstrapDialog.TYPE.PRIMARY,
  buttons: [{
    label: 'MALE',
    action: function(self, dialog) {
      BootstrapDialog.alert('You are male', function() {
        dialog.close();
      });
    }
  }, {
    label: 'FEMALE',
    type: BootstrapDialog.TYPE.DEFAULT,
    action: function(self, dialog) {
      BootstrapDialog.alert('You are female', function() {
        dialog.close();
      });
    }
  }, {
    label: 'UNKNOWN',
    type: BootstrapDialog.TYPE.DANGER,
    action: function(self, dialog) {
      dialog.close();
    }
  }]
};
BootstrapDialog.show(options);

Dialog with Event

var options = {
  title: 'Hello Dialog',
  message: '<div class="alert alert-info"></div>',
  onShow: function(dialog) {
    dialog.getModalBody().find('.alert').append('<p>show</p>');
  },
  onShown: function(dialog) {
    dialog.getModalBody().find('.alert').append('<p>shown</p>');
  }
};
BootstrapDialog.show(options);

getModalBody() is a method of the dialog, See more methods

autodestroy boolean true the dialog element would be removed from the page when closed if true
backdrop boolean true if true, click outside will close the dialog

onShow Function null

function(dialog) {...}

This event fires immediately when the show instance method is called. If caused by a click, the clicked element is available as the relatedTarget property of the event.
onShown Function null

function(dialog) {...}

This event is fired when the modal has been made visible to the user (will wait for CSS transitions to complete). If caused by a click, the clicked element is available as the relatedTarget property of the event.
onHide Function null

function(dialog) {...}

This event is fired immediately when the hide instance method has been called.
onHidden Function null

function(dialog) {...}

This event is fired when the modal has finished being hidden from the user (will wait for CSS transitions to complete).
onLoaded Function null

function(dialog) {...}

This event is fired when the modal has loaded content using the remote option.

Methods:

Name Return Description

close() The dialog instance The dialog will be hidden from the screen
setTitle(title) The dialog instance update the title of the dialog with the argument 'title'
getTitle() The dialog title as string
setMessage(message) The dialog instance Update the content of the dialog with the argument 'message', it could be a string or HTMLElement
getModalTitle() HTMLElement Get the '.modal-title' of the dialog
getModalHeader() HTMLElement Get the '.modal-header' of the dialog
getModalBody() HTMLElement Get the '.modal-body' of the dialog
